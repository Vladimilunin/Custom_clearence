## Общие инструкции для нейросети (Image-to-Image)

При использовании AI-инструментов для модификации (Stable Diffusion img2img, DALL-E edit, Midjourney /describe + /imagine):

1. **Сначала:** Загрузите исходное изображение
2. **Денойзинг:** 0.4-0.6 (сохранить форму устройства, изменить стиль и фон)
3. **Применить промпт** из соответствующего раздела выше
4. **Negative prompt:** 
```
blurry, low quality, distorted text, wrong proportions, gradient background, colored background, shadows on background, reflection, watermark
```
